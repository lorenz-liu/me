## 策略

铁鹰策略由四个不同行权价（$K_1 < K_2 < K_3 < K_4$）的期权头寸组成：

| 头寸层级 | 期权类型 | 操作类型 | 行权价 | 资金流向 | 战略作用 |
| :--- | :--- | :--- | :--- | :--- | :--- |
| **远端护翼 (外)** | **Put** | **买入 (Long)** | $K_1$ | 支出 (-) | 锁定下行极端风险（保底） |
| **主收益腿 (内)** | **Put** | **卖出 (Short)** | $K_2$ | **收入 (+)** | 赚取权利金，赌不跌破 $K_2$ |
| **主收益腿 (内)** | **Call** | **卖出 (Short)** | $K_3$ | **收入 (+)** | 赚取权利金，赌不涨破 $K_3$ |
| **远端护翼 (外)** | **Call** | **买入 (Long)** | $K_4$ | 支出 (-) | 锁定上行极端风险（封顶） |

## 初始净权利金收入公式 Net Credit

这是你开仓时**预先收到的总额**，也是该策略的**最大潜在利润 ($R$)**：

**$R = (\text{Put}_{K2} + \text{Call}_{K3}) - (\text{Put}_{K1} + \text{Call}_{K4})$**

> 由于 $K_2$ 和 $K_3$ 更接近现价，其卖出收入必然高于 $K_1$ 和 $K_4$ 的买入支出，故 $R$ 恒为正值。

## 到期损益状态表 Profit & Loss Scenarios

设到期时标的资产价格为 $S_T$，最终收益 $\Pi$ 的计算逻辑如下：

| 市场收盘价格 ($S_T$) | 最终收益 ($\Pi$) 计算公式 | 损益状态描述 |
| :--- | :--- | :--- |
| **$S_T \leq K_1$** | **$R - (K_2 - K_1)$** | **最大亏损**：触及保险上限 |
| **$K_1 < S_T < K_2$** | **$R - (K_2 - S_T)$** | **部分亏损/保本**：回吐权利金 |
| **$K_2 \leq S_T \leq K_3$** | **$R$** | **最大盈利**：所有期权价值归零 |
| **$K_3 < S_T < K_4$** | **$R - (S_T - K_3)$** | **部分亏损/保本**：回吐权利金 |
| **$S_T \geq K_4$** | **$R - (K_4 - K_3)$** | **最大亏损**：触及保险上限 |

## 关键指标 Key Metrics

* **最大利润 (Max Profit)**: $R$
* **最大风险 (Max Risk)**: $W - R$ （假设两侧价差宽度 $W = K_2 - K_1 = K_4 - K_3$）
* **上平衡点 (Upper Breakeven)**: $K_3 + R$
* **下平衡点 (Lower Breakeven)**: $K_2 - R$

## 到期后

期权到期后的处理方式，取决于该期权在到期那一刻是 **价内（ITM）** 还是 **价外（OTM）** 。 

简单来说：**价外自动失效，价内自动行权。**

| 状态 | 定义 | 结果 | 操作建议 |
| :--- | :--- | :--- | :--- |
| **价外 (OTM)** | 股价在 $K_2$ 与 $K_3$ 之间 | **自动作废** | **无需操作**。期权价值归零，你稳赚权利金 $R$。 |
| **价内 (ITM)** | 股价突破了你的行权价 | **自动行权/被指派** | **极其危险**。券商会强制按行权价买入或卖出股票。 |

如果你卖出的 $K_3$ Call 被击穿（股价 > $K_3$），在到期那一刻：
* **自动指派：** 你会被强制以 $K_3$ 的价格卖出 100 股股票。
* **风险点：** 如果你账户里没有这 100 股现货，你就会变成**持仓空头（Short Stock）**。
* **保证金压力：** 维持股票空头需要巨额保证金，如果你的账户资金不足，可能会导致爆仓或被券商强平。

虽然系统会处理到期，但 **不建议等待自动结算** 。
1. **获利了结：** 当你已经赚到了预期权利金的 80% - 90%，建议直接手动平仓（Buy to Close）。没必要为了最后那几美分去承担最后几小时的剧烈波动风险。
2. **临近行权价：** 如果股价非常接近你的卖出价（$K_2$ 或 $K_3$），**一定要手动平仓**。因为在收盘后的几分钟内，股价仍可能波动，导致原本价外的期权变成价内，从而引发意外行权（Pin Risk）。
3. **保护翅膀被击穿：** 虽然你有买入 $K_1$ 或 $K_4$ 做保险，但如果你让它们同时行权，会产生两笔高额的手续费。
