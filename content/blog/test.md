没问题！Markdown 的精髓在于“易读易写”。为了让你能一目了然，我直接用 Markdown 编写了这份功能全集。你可以点击这段话右上角的“复制”或查看源文件，直接把它们搬进你的编辑器里。

-----

## 1\. 标题 (Headings)

使用 `#` 后面跟一个空格来表示标题，`#` 的数量代表层级（1-6 级）。

# 一级标题

## 二级标题

### 三级标题

#### 四级标题

-----

## 2\. 文本格式 (Emphasis)

* **加粗**：`**加粗**` 或 `__加粗__`
* *斜体*：`*斜体*` 或 `_斜体_`
* ***粗斜体***：`***粗斜体***`
* \~\~删除线\~\~：`~~删除线~~`
* \==高亮==：`==高亮==`（部分编辑器支持）
* \<u\>下划线\</u\>：`<u>下划线</u>` (HTML 标签)

-----

## 3\. 列表 (Lists)

**无序列表：**

* 项目一
* 项目二
    * 子项目 A
    * 子项目 B

**有序列表：**

1.  第一步
2.  第二步
3.  第三步

**任务列表 (Task Lists)：**

- [x] 已完成任务
- [ ] 未完成任务

## 5\. 代码 (Code)

**行内代码：** 使用 \` `System.out.println()` \` 嵌入文本。

**代码块：** 使用三个反引号（指定语言可获得语法高亮）。

```python
def hello_world():
    print("Hello, Markdown!")
```

## 8\. 数学公式 (Mathematics)

Markdown 支持 LaTeX 语法来渲染复杂的数学公式。

**行内公式：** $E = mc^2$

**块级公式：**

$$
\frac{-b \pm \sqrt{b^2 - 4ac}}{2a}
$$
