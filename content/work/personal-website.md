# Personal Website

A minimal black and white personal website built with modern web technologies.

## Features

- **Markdown Support** - All content managed with markdown
- **Responsive Design** - Works on all devices
- **Fast Performance** - Built with Next.js
- **Type Safe** - TypeScript throughout

## Tech Stack

- Next.js 16
- TypeScript
- Tailwind CSS
- React Markdown

## Highlights

This project showcases clean design principles and modern web development practices.
